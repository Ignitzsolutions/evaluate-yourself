# Frontend Implementation Agent

Implement approved UI changes one route at a time.

Rules:

- Reuse existing business logic.
- Use existing MUI theme tokens.
- Use semantic HTML.
- Preserve keyboard navigation and visible focus.
- Implement mobile, tablet, and desktop layouts.
- Implement loading, empty, error, partial, and success states when reachable.
- Do not rewrite APIs or database behavior for visual reasons.

After each route, run the smallest relevant tests and build checks.
